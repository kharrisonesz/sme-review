Billing Report Instructions

To generate the Last Off-Platform Bill Date, run a billing search report using the following criteria:

ℹ️
How to pull your billing report:
Run a billing search for cardiac device patients billed within the last year.
Include billing from the last 91 days to present.
Filter using the CPT codes below.
Device TypeCPT Codes
Pacemaker Check93296, 93294
ICD Check93296, 93295
Loop Recorder93298
Heart Failure Diagnosis93297
Formatting Guidelines
For multiple values, separate with commas (e.g., R45, I25)
Use consistent date formatting — we recommend MM/DD/YYYY
Do not add a column for CPT codes
